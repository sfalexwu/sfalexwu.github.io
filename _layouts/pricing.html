FAQ: GETTING STARTED

  • Unzip the file and upload the index.html file into Marketo Design Studio, as a new landing page template.
  • Under your program, go to local assets, and create a new landing page and choose the landing page template you uploaded and approved. Be sure to select "Guided" in the dropdown.
  • Now in the edit view of the landing page editor, you can start adding images and replacing text.
  • Useful links: General Marketo Documentation Portal, Getting Started with Guided landing pages, editing elements, variables.

How do I pay for items on the Marketplaces?

Click the buy button for your template, and it will redirect you to gumroad to complete the purchase. We are currently using Gumroad, the secured e-commerce platform to take credit card and paypal payments.

How do I edit the template colors?

At the top of each HTML template, you will see "meta" tags which are what Marketo uses for their variables. Look for the tags with class="mktoColor", then find the hex color code (looks like: #f3f3f3) and switch it to your company's color.

How do I toggle a section OFF if I don't want it?

In the Marketo guided landing page editor, you will see on the right hand column (near the bottom) theres a variables section where you can toggle sections of the template to "SHOW" or "HIDE".

How can I edit the CSS?

1. You can edit the CSS directly inline between the <style> tags.
2. Or, you can create your own stylesheet and reference it in your <link> tag in the <head>, don't forget to put it under the bootstrap CSS stylesheet to override the default bootstrap styles.

How do I change the font?

Use the <link> tag or @import to insert custom fonts to your template. Then reference your h1,h2,h3,h4,h5,p tags in the CSS to the font-family:'Custom Font'; of your choosing.
